# AgentKit SEO

AgentKit SEO is a personal branding SEO toolkit for humans and AI agents.

It provides readable playbooks and installable agent skills for improving public career surfaces: LinkedIn profiles, GitHub profiles and repositories, ATS-friendly CVs, portfolio websites, X/Twitter profiles, and agent-readable career context files.

## Install

```bash
npx agentkit-seo install --provider codex
```

List supported providers:

```bash
npx agentkit-seo list providers
```

List available skills:

```bash
npx agentkit-seo list skills
```

## What It Covers

- LinkedIn profile optimization for recruiter search, headlines, About sections, Featured proof, skills, and experience positioning
- GitHub SEO for profile READMEs, repository descriptions, topics, pinned repos, and developer proof signals
- CV and ATS resume optimization with parser-safe structure, keywords, and evidence-based bullets
- Portfolio SEO for crawlable pages, metadata, structured data, sitemap coverage, and AI search visibility
- X/Twitter profile and posting strategy for clearer positioning and stronger profile conversion
- Agent context files that keep public rewrites grounded in verified career facts
